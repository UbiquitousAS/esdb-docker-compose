# es-cluster-compose
Docker Compose file for a secure 3 node Event Store DB cluster

#### Node1 :
- External TCP Port: 1113
- External HTTP Port: 1114

### Node2:
- External TCP Port: 2113
- External HTTP Port: 2114

### Node3:
- External TCP Port: 3113
- External HTTP Port: 3114

## Usage
```
docker-compose up
```
